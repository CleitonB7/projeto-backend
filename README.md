# Projeto Backend

Backend do projeto utilizando Node.js e SQL Server/PostgreSQL.

## Descrição

Este projeto é responsável por fornecer a API para o sistema, incluindo:  
- Conexão com o banco de dados  
- Rotas de categorias  
- Testes de conexão  
- Controllers atualizados para gerenciar entidades

## Tecnologias

- Node.js
- Express
- SQL Server / PostgreSQL
- dotenv (para variáveis de ambiente)
